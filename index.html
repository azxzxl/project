[*] Comandos para realizar varreduras [*]

nmap -vvv -sS --open -Pn -p- -T 5 |IP|

  • Esse comando realiza uma varredura intensiva e detalhada de todas as portas TCP abertas em uma máquina, fornecendo informações úteis para a identificação de possíveis vulnerabilidades.

    • nmap -vvv -sV -sC -O -p -T 5 -A --script vuln -Pn |PORTS| |IP|

  • O comando acima é usado para realizar uma varredura mais completa e detalhada em um alvo, incluindo detecção de versão de serviços, detecção de sistema operacional, teste de vulnerabilidades e execução de scripts personalizados.

    • nmap -sV --script=vuln

  • Ao executar esse comando, o Nmap realizará uma varredura de portas no alvo especificado e, em seguida, executará o script de detecção de vulnerabilidades "vuln" nos serviços encontrados nas portas abertas. O Nmap tentará identificar vulnerabilidades conhecidas com base nas versões de serviço detectadas e fornecerá um relatório com os resultados.

    • nikto -h |alvo|

  • Substitua |alvo| pelo endereço IP ou nome de domínio da aplicação web que você deseja varrer. Esse comando realizará uma varredura básica em busca de vulnerabilidades conhecidas.

    • nikto -h |alvo| -o |arquivo_de_saida|

  • Substitua |alvo| pelo endereço IP ou nome de domínio do site e |arquivo_de_saida| pelo caminho e nome de arquivo onde você deseja salvar os resultados da varredura. Essa opção é útil para revisar os resultados posteriormente ou compartilhá-los com outras pessoas.

    nc -vnz |IP| 80

  • Escaneia a porta 80 do IP

    • nc -vz www.exemplo.com 80

  • Escaneia a porta 80 do site

    • nc -vnz |IP| 20-1024

  • Escaneia da porta 20 à 1024

    • [*] ENUMERAÇÃO TOOLS [*]

    Existem várias ferramentas disponíveis, como as seguintes:

    dnsenum site.com

  • Enumeraos registros DNS de um domínio, fornecendo informações sobre os hosts, subdomínios, endereços IP e servidores de email associados.

    • nmap -p 53 --script=dns-zone-transfer --script-args dns-zone-transfer.domain=site.com

  • Realiza uma transferência de zona DNS para obter informações sobre a configuração do DNS de um domínio específico.

    • enum4linux -a |IP|

  • Executa uma enumeração em um host Windows para recuperar informações sobre usuários, compartilhamentos, grupos e políticas de segurança.

    • nbtscan -r |IP range|

  • Executa uma varredura NetBIOS em um intervalo de IP especificado para obter informações sobre hosts Windows.

    • smtp-user-enum -M VRFY -U /path/to/userlist.txt -t |IP|

  • Realiza uma enumeração de usuários SMTP para identificar usuários válidos em um servidor de email.

    • [*] Ferramentas para Brute Force [*]

    Hydra

  • Uso: hydra -l |usuário| -P |caminho/para/wordlist.txt| |IP| -t |número_de_threads| ssh
  • Exemplo: hydra -l admin -P /usr/share/wordlists/rockyou.txt 192.168.0.1 -t 16 ssh

    • Medusa

  • Uso: medusa -u |usuário| -P |caminho/para/wordlist.txt| -h |IP| -M |serviço|
  • Exemplo: medusa -u admin -P /usr/share/wordlists/rockyou.txt -h 192.168.0.1 -M ssh

    • Patator

  • Uso: patator ssh_login host=|IP| user=FILE0 password=FILE1 0=/path/to/userlist.txt 1=/path/to/passwordlist.txt
  • Exemplo: patator ssh_login host=192.168.0.1 user=FILE0 password=FILE1 0=/usr/share/wordlists/usernames.txt 1=/usr/share/wordlists/passwords.txt
